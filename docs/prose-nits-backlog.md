## 1. "Suspicion as default" repeats across three cards
Lens, Trust, and Conviction growth-edges all land on nearly the same sentence:
"discernment that protects accuracy slowly tilts into reading every difference
as deception." When the cards are read in sequence it reads templated. Root:
multiple cards key their blind-spot to the same discernment-gift template.
Fix: vary the blind-spot phrasing per card, or gate so the same blind-spot
sentence doesn't fire on 3 cards in one report.

## 2. Trust growth-edge doubles itself
Within the single Trust card growth-edge, the lead-in AND the "for your shape"
clause both state "suspicion as default … reading every difference as
deception" — the same idea twice in one paragraph. Likely a lead-in +
elaboration template that collides when both resolve to the same blind-spot.

## 3. "proces" typo in the Individual Closing Read
The Individual report's Closing Read (the appended synthesis) reads "the way you
**proces** the world" — missing the 's'. The Guide's "A Synthesis" has it
correct as "process", so the typo is in the Individual closing-read composition
path specifically. Grep for "proces" (no trailing s) in the
fiftyDegreeIndividual / renderMirror closing-read composer.
